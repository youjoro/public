Soil Condition Monitoring System using Arduino

     Created a system where the arduino is able to monitor the factors that affect soil health, these factors are:
Soil Moisture,
Humidity,
Temperature,
and Soil Type
These data are then sent to the mobile and web app which is able to show the data into a graphical chart, these data can also be used for predicting soil health or to determine which plants are a perfect match to the soil.

dServ: Home Service Booking Web App

     Created a web app where clients and home service providers meet, clients can find home services within their immediate area or within a city, chat with the provider and book their services.


Home Service Providers can register with the web app, set where they're located and add services that they provide. They can edit, and manage their services to either make it clear what they provide or make the service more appealing. The web app also allows Home Service Providers to manage client requests, they can accept, decline and change the date of the appointment.
